# Blood Management System - Database

## Overview

This folder contains the database design and implementation for the Blood Management System. It includes the SQL schema, Entity Relationship Diagram (ERD), and documentation of the database structure.

---

## Project Files

- `schema.sql` – SQL script for creating the database.
- `ERD FINAL.pdf` – Entity Relationship Diagram (ERD).
- `README.md` – Documentation for the database module.

---

## Database Tables

- Users
- Blood Banks
- Donation Slots
- Appointments
- Blood Inventory
- Blood Requests
- Blood Request Units
- Notifications

---

## Features

- User management
- Blood bank management
- Donation appointment scheduling

## Notes

The database is designed to support the core functionalities of the Blood Management System, including:

- User management
- Blood bank management
- Appointment scheduling
- Blood inventory management
- Blood request processing
- Notification management

The database schema is normalized to maintain data consistency, reduce redundancy, and ensure efficient relationships between entities.
